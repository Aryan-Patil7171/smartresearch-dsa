# Screenshots

> Add your screenshots to this `docs/` folder and link them below.

## How to take screenshots

**Terminal app:**
```bash
clang++ main.cpp -std=c++17 -o research
./research
```
Take a screenshot after running a few menu options (e.g., Display Researchers, Sort Projects, BFS).

**React frontend:**
```bash
# Terminal 1
node api/server.js

# Terminal 2
cd frontend/react-app && npm run dev
```
Open http://localhost:5174 and take screenshots of each page.
